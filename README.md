# anomaly-dashboard
A dashboard for anomaly detectioon

A frontend dashboard for visualizing anomaly and suspicious activity detection in surveillance systems.
The application simulates real-time monitoring of camera feeds, detected threats, and operator actions.

This project focuses on UI/UX design and layout, serving as a foundation for integrating real-time video streams and AI-based detection models in the future.
